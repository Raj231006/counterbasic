Score Counter Web App

A simple web-based score tracker for two teams: Home and Guest. Users can add points, apply penalties, or reset scores, all in real time. Built using HTML, CSS, and JavaScript.

Features

Increase Home or Guest score by 1, 2, or 3 points.

Apply a -1 penalty to both teams.

Reset scores to 0 using the Restart button.

Visually styled and responsive layout.

Technologies

HTML – Page structure and content.

CSS – Layout, styling, and visual design.

JavaScript – Handles score changes and button functionality.

How to Use

Open index.html in any modern browser.

Click +1, +2, or +3 to add points to the respective team.

Click -1 to apply a penalty to both teams.

Click RESTART to reset both scores to zero.
